Download Voice Assistant Chrome Extension

Installation Instructions

  1. Download the extension by clicking the button above.
  2. Extract the .zip file to a folder on your computer.
  3. Open Google Chrome and go to "chrome://extensions".
  4. Enable "Developer mode" by toggling the switch in the top-right corner.
  5. Click the "Load unpacked" button and select the folder where you extracted the extension files.
  6. Your extension should now be installed and visible in the extensions list.
  7. Right click the extension in the toolbar, click "View Web Permissions" and set microphone to "Always."
